# แหล่งข้อมูล

โปรเจกต์นี้ใช้ชุดข้อมูลสาธารณะจากหน่วยงานภาครัฐไทย ที่เข้าถึงได้ผ่าน data.go.th และแหล่งข้อมูลทางการของแต่ละหน่วยงาน ไฟล์ทั้งหมดดาวน์โหลดมาเก็บไว้ใน `data/raw/` และนำเข้าโดยตรงโดยไม่มีการแก้ไขเนื้อหาต้นฉบับ

## ชุดข้อมูลหลัก: สถาบันบริหารจัดการธนาคารที่ดิน (LABAI)

### 1. จำนวนผู้ประสงค์ใช้ที่ดินและเจ้าของที่ดินที่ลงทะเบียน

| รายละเอียด | ข้อมูล |
|---|---|
| **ชื่อชุดข้อมูล** | จำนวนข้อมูลผู้ประสงค์เข้าใช้ประโยชน์ในที่ดินและเจ้าของที่ดิน ที่ลงทะเบียน ของโครงการตัวกลางที่ดิน |
| **หน่วยงาน** | สถาบันบริหารจัดการธนาคารที่ดิน (LABAI) |

| **รูปแบบ** | JSON |
| **จำนวนแถว** | 77 (หนึ่งแถวต่อจังหวัด) |
| **Source URL** | https://data.go.th/dataset/65_dataset_11_03 |
| **Resource URL** | https://app.labai.or.th/reportclm/owner_want_Json |
| **ใช้ทำอะไร** | เป็นข้อมูลหลักของการวิเคราะห์ทั้งโปรเจกต์ ใช้เปรียบเทียบจำนวนผู้ประสงค์ใช้ที่ดิน (`count_want_land`) กับจำนวนเจ้าของที่ดิน (`count_owner_land`) ระดับจังหวัด |
